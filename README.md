# 🤖 Lattinaa AI

Lattinaa è un chatbot AI locale scritto in Python che utilizza modelli **GGUF** tramite **GPT4All**.
Funziona completamente in locale sul tuo computer.

## ✨ Funzioni

* chatbot AI locale
* memoria delle conversazioni
* aggiornamenti automatici
* interfaccia grafica
* download automatico del modello

---

## 📦 Installazione

1. Scarica il progetto
2. Estrai lo ZIP
3. Avvia:

```
installer.exe
```

L'installer scaricherà automaticamente il modello AI.

---

## 🚀 Avviare Lattinaa

Dopo l'installazione esegui:

```
main.py
```

Oppure il file eseguibile se presente.

---

## 🧠 Modello AI

Lattinaa utilizza:

```
qwen2.5-coder-7b-instruct-q4_0.gguf
```

Il modello viene scaricato automaticamente dall'installer.

## 🔄 Aggiornamenti

Lattinaa controlla automaticamente se esistono nuove versioni tramite **GitHub Pages**.
